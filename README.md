# Lab System

نسخة محسنة من منظومة المختبر تعتمد على:

- خادم Node.js واحد فقط
- قاعدة بيانات SQLite بدل JSON في الذاكرة
- مصادقة خادمية Token-based
- Pagination وSearch لسجلات المرضى
- ترحيل تلقائي من ملفات JSON القديمة إلى SQLite عند أول تشغيل

## متطلبات التشغيل

- Node.js 24 أو أحدث

## الإعداد السريع

1. انسخ القيم من `.env.example` إلى متغيرات البيئة المناسبة لديك.
2. غيّر `ADMIN_PASSWORD` إلى كلمة مرور قوية قبل التشغيل في بيئة الإنتاج.
3. شغّل الخادم:

```powershell
npm start
```

## أول تشغيل

- إذا لم تحدد `ADMIN_PASSWORD` فسيتم إنشاء كلمة مرور مؤقتة قوية تلقائياً.
- ستُكتب بيانات الدخول المؤقتة في الملف `storage/bootstrap-admin.txt`.
- بعد تسجيل الدخول لأول مرة، غيّر كلمة المرور ثم احذف هذا الملف.

## التخزين

- قاعدة البيانات: `storage/lab_system.sqlite`
- النسخ الاحتياطية لبيانات JSON القديمة: `storage/backups/`

## ملاحظات مهمة

- واجهة PHP القديمة لم تعد هي المسار الرئيسي لهذا الإصدار.
- واجهات النتائج العامة ما زالت متاحة بدون تسجيل دخول عبر `/api/public-results/:token`.
- واجهات الإدارة الداخلية تتطلب Token مصادقة من `/api/auth/login`.
