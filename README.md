# موقعي على GitHub Pages 🚀

موقع ويب حديث ومتجاوب مبني بـ HTML وCSS وJavaScript خالص، جاهز للنشر على **GitHub Pages**.

## 🌟 مميزات الموقع

- تصميم عصري بألوان داكنة جذابة
- دعم كامل للغة العربية (RTL)
- تجاوب مع جميع أحجام الشاشات
- رسوم متحركة سلسة عند التمرير
- عداد إحصائيات تفاعلي
- نموذج تواصل تفاعلي
- شريط تنقل ثابت مع تأثير الشفافية

## 📁 هيكل المشروع

```
├── index.html      ← الصفحة الرئيسية
├── style.css       ← ملف التنسيق (CSS)
├── script.js       ← ملف التفاعل (JavaScript)
└── README.md       ← هذا الملف
```

## 🚀 النشر على GitHub Pages

1. اذهب إلى إعدادات المستودع → **Settings**
2. من القائمة الجانبية اختر **Pages**
3. تحت **Branch** اختر الفرع الرئيسي (`main` أو `master`)
4. اضغط **Save**
5. بعد دقيقة سيكون الموقع متاحاً على:  
   `https://username.github.io/repo-name/`

## 🎨 التخصيص

افتح ملف `style.css` وعدّل المتغيرات في أعلى الملف:

```css
:root {
  --primary: #6c63ff;   /* اللون الرئيسي */
  --secondary: #ff6584; /* اللون الثانوي */
}
```
